rovercode

Powerful block-based programming for educational robots

$ git clone https://git.io/rovercode.git && sudo ./rovercode/setup.sh

Free and open-source

Never paid and always improving. Some modules MIT; some GPLv2. Check out the source, feature-tracking, and wiki docs on Github.

Web-app centered

Works with just a browser, on any device. Built on Blockly.

Choose your hardware

Host from a C.H.I.P, Raspberry Pi Zero, or a BeagleBone Black robot. rovercode automatically detects your platform and uses the appropriate HAL.

Try it! Improve it!

The current release of rovercode is very functional, and devs like you are actively adding cool new features. Try it out in the live demo, then pull it down and help make it even better!
