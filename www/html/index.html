rovercode

Powerful block-based programming for educational robots

$ git clone https://git.io/rovercode.git

Choose your hardware

Host from a C.H.I.P, Raspberry Pi, or a BeagleBone robot. rovercode automatically detects your platform and uses the appropriate HAL.

Web-app centered

Works with just a browser, on any device. Built on Blockly.

Free and open-source

Never paid and always improving. Some modules MIT; some GPLv2. Check out the source, feature-tracking, and wiki docs on Github.

Try it, Improve it

The current release of rovercode is functional. Try it out in the live demo, then pull it down and help make it even better

The mission of the rovercode.org community is to provide educators with a capable, easy-to-use, block-based programming environment that can be hosted on low-cost robotics hardware.
