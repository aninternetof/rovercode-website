rovercode

Powerful block-based programming for educational robots

$ git clone https://git.io/rovercode.git && sudo ./rovercode/setup.sh

Free and open-source

Never paid and always improving. Some modules MIT; some GPLv2.

Web-app centered

Works with just a browser, on any device. Built on Blockly.

Choose your hardware

Host from a C.H.I.P, Raspberry Pi Zero, or a BeagleBone Black robot.

Live Demo
The mission of the rovercode.org community is to provide educators with a capable, easy-to-use, block-based programming environment that can be hosted on low-cost robotics hardware.
